# Chat Archives

This directory contains saved chat logs from GitHub Copilot sessions.

## Naming Convention
- `CHAT_LOG_YYYYMMDD_HHMMSS.md` - Auto-generated timestamp
- Custom names can be used for specific sessions

## How Files are Created
- Using `save_chat.sh` script in root directory
- Manually created and committed
- Automatically committed to git when using the script

## Contents
Each chat log contains:
- User questions and requests
- AI responses and code
- Troubleshooting steps
- Solutions implemented
- File changes made

## Purpose
- Reference for problem-solving approaches
- Documentation of development decisions
- Backup of important conversations
- Knowledge base for future issues
